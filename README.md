Shopify E-Commerce Frontend Project

This project is a responsive e-commerce website built using HTML, CSS, and JavaScript. The application simulates a basic online shopping interface where users can browse products, view collections, and search for items.

Key Features:

* Responsive navigation bar with mobile side menu
* Homepage with hero section, services, and product highlights
* Collections page displaying products in a flexible grid layout
* Real-time product search functionality using JavaScript
* Product cards with images and names
* Newsletter subscription section
* Footer with brand details and social media icons
* Fully responsive design using Flexbox and media queries

Technologies Used:
HTML5 for page structure
CSS3 for layout, styling, and responsive design
JavaScript for DOM manipulation and interactive features

Project Pages:
Home page – displays featured sections and product highlights
Collections page – shows all products with search filtering
Contact page – provides basic contact interface

This project demonstrates front-end development concepts such as responsive layout design, UI component structuring, event-driven JavaScript, and dynamic product filtering.
